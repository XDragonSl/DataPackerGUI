Use "File->New", or "Ctrl+N", or "N" if menu in focus, to choose files to pack and press "Create pack" button.


Use "File->Open", or "Ctrl+O", or "O" if menu in focus, to choose file to unpack and press "Extract all" button.


Use "File->Exit", or "Ctrl+E", or "E" if menu in focus, to quit.


Use "Help->Help Contents", or "Ctrl+H", or "H" if menu in focus, to show the help page.


Use "Help->About Data Packer", or "Ctrl+A", or "A" if menu in focus, to show the information page.
